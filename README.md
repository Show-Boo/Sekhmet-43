### 🎮조작법(Control)
**이동(Move)**: `W,A,S,D` <br>

**카메라 이동(Camera)**: `🖱 Mouse` <br>

**달리기(Sprint)** : `Left Shift` <br>

**점프(Jump)**: `Space bar`<br>

**손전등(FlashLight)**: `F`<br>

**퀘스트(Quest)**: `Q`<br>
